# Taller Integrado: DevOps + Low Code / No-Code  
Carrera: Desarrollo de Software  
Materia: Tendencias Actuales  
Estudiante: Luis Chango  

## Objetivo
Comprender cómo se integran procesos DevOps (automatización, CI/CD, colaboración) con herramientas Low Code / No-Code en un flujo de desarrollo moderno.

---

## Parte 1: DevOps – Automatización y CI/CD

### ✔ Aplicación creada
Se construyó una aplicación simple en Python (`app.py`) que retorna un mensaje.  
Esta app se usa para demostrar cómo funciona un pipeline CI/CD básico.

### ✔ Pipeline implementado
Se creó un workflow en GitHub Actions (`ci-cd-demo.yml`) que:
- Verifica el código
- Ejecuta la aplicación
- Muestra un pipeline funcional automatizado

---

## Parte 2: Low Code / No-Code
Se investigó y documentó (en `lowcode-info.md`) cómo herramientas LC/NC permiten:
- Crear aplicaciones rápidas sin programar desde cero
- Integrarse con flujos DevOps
- Automatizar procesos empresariales

---

## Conclusión
Este taller demuestra cómo DevOps y Low Code/No-Code trabajan juntos para acelerar el desarrollo, mantener calidad y mejorar la entrega continua.
