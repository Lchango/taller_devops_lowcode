# Low Code / No-Code

## ¿Qué es Low Code?
Conjunto de herramientas que permiten crear aplicaciones con mínima programación, usando componentes visuales.

### Ejemplos:
- Mendix  
- OutSystems  
- Retool  
- PowerApps  

## ¿Qué es No-Code?
Permite crear soluciones completas sin escribir código.

### Ejemplos:
- Zapier  
- Notion  
- Glide  
- AppSheet  

---

## Ventajas
- Desarrollo rápido  
- Fácil mantenimiento  
- Menor costo  
- Integración con APIs y servicios DevOps  

## Relación con DevOps
- Automatizan tareas repetitivas  
- Facilitan despliegues  
- Permiten integraciones CI/CD  
- Mejoran colaboración entre áreas
